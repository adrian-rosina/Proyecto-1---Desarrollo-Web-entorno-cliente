# Proyecto-1---Desarrollo-Web-entorno-cliente
Este repositorio hace referencia al proyecto 1 de la asignatura de desarrollo web entorno cliente


# Enunciado:

TechMarket es una tienda online de tecnología que ha detectado un problema
grave:
Los usuarios entran en la web… pero no compran.
Tras analizar el comportamiento, el equipo ha identificado varios fallos:
Los productos no se muestran de forma clara
No existe un sistema para filtrarlos
No hay forma de visualizar lo que el usuario quiere comprar
El formulario de compra genera errores constantes
El equipo técnico ha decidido rehacer la interfaz completamente en entorno
cliente, creando una versión funcional que permita validar si la experiencia de
usuario mejora.
Tu tarea como desarrollador/a frontend es construir esta nueva versión.
Desarrollar una página web que permita:
• Mostrar productos dinámicamente
• Filtrarlos por categoría
• Añadir productos a un carrito
• Mostrar el contenido del carrito
• Validar un formulario de compra
Todo debe realizarse en HTML, CSS y JavaScript, ejecutándose en el navegador.
Debes trabajar con una estructura de productos como la siguiente:
const productos = [
{ id: 1, nombre: "Teclado", precio: 30, categoria: "perifericos" },
{ id: 2, nombre: "Ratón", precio: 20, categoria: "perifericos" },
{ id: 3, nombre: "Monitor", precio: 200, categoria: "pantallas" },
{ id: 4, nombre: "Portátil", precio: 800, categoria: "ordenadores" }
];
